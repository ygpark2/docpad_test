# 수행 일지

오늘은 너무 않 되는 것에만 빠지는 것 같아 좀 가벼운 마음을 가졌습니다. 너무 않 되는 것에만 집중하다 보니 나를 자책하게 되는 것 같고, 그러다 보니 수행이 조금씩 무거운 느낌이었습니다. 법륜스님께서 항상 가볍게 하라는 말씀을 떠올리며 무거운 저 자신을 보니 아무래도 수행을 잘 못하고 있는것 같아 오늘은 무거운 마음을 털어버리는 것에 집중했습니다.
